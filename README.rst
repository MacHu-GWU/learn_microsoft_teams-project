Install
------------------------------------------------------------------------------

``learn_microsoft_teams`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-microsoft-teams

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-microsoft-teams
